## System Skill
A simple skill to shutdown/restart mycroft by voice.

## Description
This skill allows the user to turn off (or restart) Mycroft gracefully, with a voice conformation (yes).

## Examples
 - "Shut down"
 - "Power off"
 - "Reboot"
 - "Restart"
