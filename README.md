## System Skill
A simple skill to shutdown/restart system by voice.

## Description
This skill allows the user to turn off (or restart) system (tested on Rpi) gracefully, with a voice confirmation (yes).

## Examples
 - "system poweroff"
 - "system shutdown"
 - "system die"
 - "system off"
 - "system reboot"
 - "system restart"
